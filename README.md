# Santander Bootcamp Back-end Java (em parceria com a DIO)

Repositório com os exercícios práticos e o projeto final desenvolvidos durante o Bootcamp Back-end Java do Santander, em parceria com a DIO, cobrindo fundamentos de Java, Spring Boot, persistência de dados e boas práticas de desenvolvimento back-end.

## 🎯 Objetivo

Registrar minha evolução durante o bootcamp, aplicando conceitos de:

- Lógica de programação e fundamentos de Java
- Orientação a Objetos
- Spring Boot
- Persistência de dados (JPA/Hibernate)
- APIs RESTful
- Boas práticas de código

## 📁 Estrutura

- `/exercicios` — exercícios práticos de cada módulo
- `/projeto-final` — projeto final do bootcamp
